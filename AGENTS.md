# AGENTS Instructions

- For commits in this repository, follow the **Commit Message Convention** in [`CONTRIBUTING.md`](./CONTRIBUTING.md).
- UI components should use the `shadcn-vue` component library whenever an equivalent component exists. Avoid introducing raw HTML controls like `<select>`, `<button>`, `<input>`, and tabs in feature code unless there is no suitable `shadcn-vue` component.
- For icons, use the existing Iconify/UnoCSS icon classes (e.g. `i-lucide-*`, `i-solar-*`) instead of adding dedicated Vue icon component packages.
- If instructions conflict, system / developer / user instructions take precedence.
